spring boot 无法启动： ComponentScan of the default package

发表于 2016-06-22 分类于 bug处理记录

报错内容   

原因：spring boot 的启动类没有直接放在了 java 文件夹下，导致spring boot 无法找到它

解决办法  将spring boot 的启动类移动到某个具体的包下面（没有包则新建一个）
